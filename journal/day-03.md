## 核心知识点

### 数据工具
- `yf.download(ticker, period='1y', auto_adjust=True)` 下载 OHLCV + Adj Close
- yfinance 默认返回 MultiIndex columns（2024 版起），用 `get_level_values(0)` 拍平
- 金融 DataFrame 的标准结构：index = 日期（DatetimeIndex），columns = 价格字段

### 收益率
- 简单收益：`pct_change()` = (P_t - P_{t-1}) / P_{t-1}
- 对数收益：`np.log(P_t / P_{t-1})`
- 简单收益用于加总（组合加权），对数收益用于累积（时间叠加）

### 风险调整收益指标
- **夏普比率** = (年化收益 - 无风险) / 年化波动率
- **Calmar 比率** = 年化收益 / |最大回撤|
- **Sortino 比率** = (年化收益 - 无风险) / 下行波动率
- 年化规则：收益 × 252，波动 × √252

### 最大回撤
- `cum_value = (1+r).cumprod()`
- `running_max = cum_value.cummax()`
- `drawdown = (cum_value - running_max) / running_max`
- MDD = drawdown.min()

### 2. 改 `risk_free=0.3` 夏普没变
因为只改了变量，没把它放进公式。**Python 不会"自动代入"同名变量**。
公式里必须显式写 `(ann_return - risk_free) / ann_vol`。

### 3. `0.3` vs `0.03` 的单位混淆
无风险利率是年化小数形式，`0.03 = 3%`，`0.3 = 30%`。金融计算里永远要警惕单位。

## 今日亮点实验：√T 年化公式的边界

**问题**：`σ × √252` 年化公式依赖 IID 假设。假设失效时，公式会错多少？

**实验**：
- 模拟 IID 收益 → 公式 vs 真实误差 < 1% ✅
- 模拟 AR(1) ρ=+0.3 → 公式**低估真实风险约 30%**
- 模拟 AR(1) ρ=-0.3 → 公式**高估真实风险**
- SPY 日收益 lag-1 自相关 ≈ 0 → 对 SPY 来说标准夏普是可靠的

**结论**：
- 买入持有型策略（像 SPY）→ 标准夏普可信
- 动量 / 趋势跟踪策略 → 标准夏普低估真实风险
- 均值回归策略 → 标准夏普高估真实风险
- 这就是为什么 López de Prado 要提出 Deflated Sharpe Ratio

## 今天的思考深度：市场自相关的底层逻辑

今天问了一个让自己很兴奋的问题：**为什么市场会有自相关？**
和 Claude 讨论后，发现它源于三层机制的叠加：

### 第 1 层：信息传播的金字塔
信息从"知情者 → HFT → 机构 → 分析师 → 散户 → 被动基金"层层下渗，
每一层需要的时间不同。这直接决定了：
- 毫秒级：做市商套利 → 负自相关
- 周-月级：信息浸润未完成 → 正自相关（动量）

### 第 2 层：人类行为偏差
- **锚定效应**：分析师慢慢上调目标价 → 价格渐进上涨
- **处置效应**：短期抑制上涨，中期加速 → 3-12 月动量
- **过度反应**：短期偏离过大 → 短期反转
- **可得性启发**：情绪周期 → 长期均值回归

### 第 3 层：结构性资金流
- 被动基金机械买卖 → 强化动量
- CTA 趋势跟踪 → 自我实现的动量
- 风险平价 → 波动率反馈循环

### 最深的一个认知
> **市场是生态系统，不是机器。**  
> 每个规律被发现后会被交易，被交易后规律本身会改变。  
> 顶级 quant 找的不是"聪明规律"，是"结构性规律"——  
> 那些深植于市场结构、不会被套利消失的规律。

这对应 Andrew Lo 的 Adaptive Markets Hypothesis。
书单加一条：Andrew Lo, *Adaptive Markets* (2017)。
